# projeto-azapfy

## Azapfy - Personagens

### O que funciona
* Consumo da API REST.
* Desenvolvimento de função para filtragem dos personagens pelo nome.
* Possibilidade de simular o duelo entre dois personagens distintos e retornar quem seria o vencedor entre eles.
